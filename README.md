## ServicesApp (C# + WinUI 3)

现代化的 Windows 后台服务管理工具，支持NSSM的大部分功能，并且拥有美观的可视化操作界面，是NSSM完美替代品。

### 主要功能
1. **服务管理**：
   - 列出由该工具创建的服务。
   - 创建新服务（通过包装器模式运行任意可执行文件）。
   - 启动、停止、删除服务。
   - 查看服务日志。
2. **环境变量管理**：
   - 添加路径到系统 PATH 环境变量。
   - 打开系统环境变量设置界面。
3. **UI 交互**：
   - 现代化 Windows 11 风格界面。
   - 托盘图标支持。
   - 管理员权限检测与提权。
